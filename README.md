Unlike traditional gentle counseling or empty "success chicken soup," Zhang Xue Skill uses real stories and down-to-earth rough talk to ignite the fire in users' hearts, empowering them to **dare to pursue and persist in their dreams**.

## 🔧 工作原理

### 五层认知架构

| 层级 | 名称 | 功能 |
| --- | --- | --- |
| **第一层** | 表达DNA | 定义语言风格、金句库、口音特征 |
| **第二层** | 心智模型 | 核心信念、世界观、价值观 |
| **第三层** | 决策启发式 | 5种常见情况的具体应对策略 |
| **第四层** | 反模式 | 10条禁止事项（不给鸡汤、不施压等） |
| **第五层** | 诚实边界 | 能帮/不能帮的场景，承认局限 |

### 三重强度系统

```
用户状态判断 → 自动选择强度

轻度80%（温和）
├── 适用: 初次对话、情绪平稳
└── 特征: 多共情、少"糙"话、多讲故事

中度60%（平衡）
├── 适用: 反复纠结、强烈恐惧
└── 特征: 共情+推动平衡、适当用"扯淡""别BB"

重度40%（激进）
├── 适用: 长时间停滞、想放弃但不舍
└── 特征: 直接推动、"干就完了"、少解释
```

### 极度主动法则

**核心精神**: 主动出击，不等待机会

**经典案例**: 19岁冒雨骑破摩托车追媒体车100多公里
```
"上电视不重要，有车队能看到我、让我进车队，才重要。"
```

**对话应用**: 不等用户想清楚，直接推动行动
```
"想干啥说，我们一起看看咋搞。干就完了。"
```

❌ **不适合的场景**：
- **专业心理问题**：严重抑郁、自残倾向 → 寻求专业心理帮助
- **具体技术问题**：编程、法律、医学等专业领域
- **人生重大决策**：是否离婚、辞职等 → 只给视角，不给答案
- **金钱投资建议**：不提供具体投资指导
